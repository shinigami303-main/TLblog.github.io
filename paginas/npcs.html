Curiosidades

Para um npc se instalar em determinado bioma, o jogador precisa construir casas para eles. Não precisa ser uma casa bem elaborada, uma casinha de 5x10 já é o necessário.

Ao combinar 2 ou mais npcs de mesmo bioma em seu respectivo bioma, os mesmos vendem pliwos, que são utilizados para se locomover de bioma em bioma, como um sistema de teletransporte
